Sobre o Projeto

Este projeto foi desenvolvido por estudantes do curso do 7°semestre de Ciência da Computação como parte de nossa avaliação das 3 matérias: Inteligência Artificial, Virtualização de servidores e Tópicos avançados de Computação.

Objetivo

Desenvolver uma ferramenta que utiliza algoritmos genéticos para otimizar hiperparâmetros de redes neurais convolucionais (CNNs), especificamente para o dataset CIFAR-10.

Equipe

  • Elian Gomes
  • Jossylyn Casseb
  • Lia Nicole Costa

Tecnologias Utilizadas

  • Python com PyTorch para o backend
  • FastAPI para a API REST
  • HTML, CSS e JavaScript para o frontend
  • Docker para containerização
